# 🎓 SUSS – Student Utility Support System

An all-in-one student productivity web application designed to manage academic and personal tasks efficiently.

---

## 🚀 Features

* 📊 Expense Tracker
* 📝 Notes Management
* 📅 Planner
* 📄 Resume Section
* 🔐 User Authentication (Login/Register)
* 📌 Clean Dashboard UI

3. Run the project:
